# Karv & Associates - Chartered Accountants Website

A professional website for **Karv & Associates**, a fictional Chartered Accountancy firm based in India offering comprehensive financial services.

## Services Offered

- **Income Tax Services** - ITR Filing, Tax Planning, TDS Returns
- **GST Services** - Registration, Return Filing, Audit, Refund
- **Audit & Assurance** - Statutory Audit, Tax Audit, Internal Audit
- **Company Registration** - Private Limited, LLP, OPC, Partnership
- **Accounting & Bookkeeping** - Financial Statements, Book Keeping
- **Business Advisory** - Strategic Planning, Financial Consulting

## Pages

- `index.html` - Main landing page
- `blogs/gst-services.html` - GST services details
- `blogs/tax-services.html` - Income tax services with calculator
- `blogs/audit-services.html` - Audit & assurance services
- `blogs/company-registration.html` - Company registration packages
- `blogs/accounting-services.html` - Accounting & bookkeeping services
- `blogs/advisory-services.html` - Business advisory services

## Tech Stack

- HTML5
- CSS3 (No frameworks - custom styling)
- Vanilla JavaScript
